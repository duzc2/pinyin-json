# pinyin-json

JSON data for Chinese Pinyin

* Pinyin table
  ```json
  ["a", "o", "e", "ê", "ai", "ei", "ang", "eng", "er", "yi", "..."]
  ```
* Pinyin-Chinese table
  ```json
  {
    "a": ["啊", "阿"],
    "he": ["和", "喝", "何", "合", "贺"]
  }
  ```
* Chinese-Pinyin table
  ```json
  {
    "啊": ["a"],
    "阿": ["a", "e"],
    "和": ["he", "huo"],
    "...": "..."
  }
  ```
* Words list fetched from Chinese Wikipedia, Chinese Wikitionary
  ```json
  ["我们", "世界", "Linux", "V2火箭"]
  ```
